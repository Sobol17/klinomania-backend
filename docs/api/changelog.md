# API Changelog

## Unreleased

- Добавлен базовый контракт API v1.
- Добавлена клиентская авторизация по телефону и 4-значному коду.
- Клиентский OTP генерируется случайно; временная dev-заглушка `1111` доступна только при явном включении.
- Добавлена авторизация клинера по телефону и 6-значному коду.
- Добавлена временная dev-заглушка кода клинера `111111`.
- Добавлены endpoints услуг, профилей, клиентских заявок и работы клинера.
- Добавлено редактирование клиентского профиля: имя, email, адрес и разрешения на PUSH/email-рассылки.
- Добавлен `POST /client/push-token` для регистрации FCM-токена и статусные PUSH-уведомления по заявкам.
